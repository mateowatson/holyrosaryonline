About

HolyRosaryOnline.com is a website meant to help people pray the Rosary. It is an open source project maintained by web developer Matt Watson. Different options are available in Settings.

Click the beads as you go along, or click anywhere inside the rosary to go to the next bead.

Email matt@mattwatson.org for questions or leave an issue on the GitHub page.

Sources

The texts of the prayers were taken in fair use from various sources.

Artwork

Artwork has been added by default as an aid to contemplation. You may find out more about these images on the artwork page.
